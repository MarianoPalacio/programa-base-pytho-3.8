Modificaciones
Ejecutar el progrma desde el archivo unidad_8.py (Desde sublime text3, presionar Ctrl+b)
Es un programa base para practicar alta,  baja y modificacion de registros dentro de una base de datos Sqlite3.

Detalle de modificaciones durante el desarrollo del mismo a continuación.
Se aplicó peewee ORM en forma de base sqlite3
Se eliminaron botones que se consideraban redundantes.
Se eliminaron campos, etiquetas destinadas a título y descripción.Ya que el popup del botón 'Guardar' cumple con este objetivo.

Se eliminó el botón ttk para mostrar resultados, ya que mientras la base exista, permanecen en pantalla los datos almacenados.
Se acondicionó función mostrar dentro de archivo unidad_8.py
Se corrigieron errores surgidos por dichas modificaciones.
Se eliminaron todos los "print" que figuraban en consola.
Se agregó un botón de ayuda para orientar al usuario en como utilizar los distintos botones.
Se eliminó el botón crear base ya que la misma se crea automáticamente.
